# Measure Theory and Integration

Measure theory provides a rigorous foundation for integration that extends beyond the limitations of the Riemann integral. By shifting focus from partitioning the domain to partitioning the range and defining a formal "measure" on sets, it allows for the integration of highly discontinuous functions and functions defined on complex spaces.

## ã . Foundations: Measurable Spaces and $\sigma$-Algebras

A **measurable space** $(X, \mathcal{A})$ consists of a set $X$ and a collection of subsets $\mathcal{A}$ that form a structure for consistent measurement.

### 1.1 $\sigma$-Algebras

A $\sigma$-algebra $\mathcal{A}$ on a set $X$ is a collection of subsets of $X$ satisfying:

1.  **Closure under Complementation:** If $A \in \mathcal{A}$, then $A^c = X \setminus A \in \mathcal{A}$.
2.  **Closure under Countable Union:** If $\{A_i\}_{i=1}^{\infty} \subseteq \mathcal{A}$, then $\bigcup_{i=1}^{\infty} A_i \in \mathcal{A}$.
3.  **Non-emptiness:** $X \in \mathcal{A}$.

The Borel $\sigma$-algebra $\mathcal{B}(X)$ is the $\sigma$-algebra generated by all open sets of a topological space $X$.

### 1.2 Measures

Given a measurable space $(X, \mathcal{A})$, a **measure** $\mu: \mathcal{A} \to [0, \infty]$ is a function satisfying:

1.  **Non-negativity:** $\mu(A) \ge 0$ for all $A \in \mathcal{A}$.
2.  **Null Empty Set:** $\mu(\emptyset) = 0$.
3.  **Countable Additivity:** For any sequence of pairwise disjoint sets $\{A_i\}_{i=1}^{\infty} \subseteq \mathcal{A}$,
    $$\mu\left(\bigcup_{i=1}^{\infty} A_i\right) = \sum_{i=1}^{\infty} \mu(A_i)$$

### 1.3 Carathûˋodory Construction

An **outer measure** $\mu^*$ is defined for all $E \subseteq X$ by:
$$\mu^*(E) = \inf \left\{ \sum_{i=1}^{\infty} \mu(A_i) : E \subseteq \bigcup_{i=1}^{\infty} A_i, A_i \in \mathcal{A} \right\}$$

The **Carathûˋodory Extension Theorem** states that the collection of sets $E$ satisfying the condition $\mu^*(A) = \mu^*(A \cap E) + \mu^*(A \cap E^c)$ for all $A \subseteq X$ forms a $\sigma$-algebra, and the restriction of $\mu^*$ to this $\sigma$-algebra is a complete measure.

## ãÀ. The Lebesgue Integral

The Lebesgue integral is constructed in stages over a measure space $(X, \mathcal{A}, \mu)$.

### 2.1 Measurable Functions

A function $f: X \to \mathbb{R}$ is **measurable** if for every Borel set $B \subset \mathbb{R}$, the preimage $f^{-1}(B) \in \mathcal{A}$.

### 2.2 Integral Construction

1.  **Indicator Functions:** For $A \in \mathcal{A}$, $\int_X \mathbf{1}_A d\mu = \mu(A)$.
2.  **Simple Functions:** For $\phi = \sum_{i=1}^n c_i \mathbf{1}_{A_i}$, $\int_X \phi d\mu = \sum_{i=1}^n c_i \mu(A_i)$.
3.  **Non-negative Functions:** For $f \ge 0$, $\int_X f d\mu = \sup \{ \int_X \phi d\mu : 0 \le \phi \le f, \phi \text{ is simple} \}$.
4.  **General Functions:** $\int_X f d\mu = \int_X f^+ d\mu - \int_X f^- d\mu$, provided at least one integral is finite.

### 2.3 Convergence Theorems

**Monotone Convergence Theorem (MCT):** If $0 \le f_1 \le f_2 \le \dots$ and $f_n \to f$ pointwise, then $\lim_{n\to\infty} \int_X f_n d\mu = \int_X f d\mu$.

**Fatou's Lemma:** For non-negative measurable $f_n$, $\int_X \liminf_{n\to\infty} f_n d\mu \le \liminf_{n\to\infty} \int_X f_n d\mu$.

**Dominated Convergence Theorem (DCT):** If $f_n \to f$ a.e. and $|f_n| \le g$ for some integrable $g$, then $\lim_{n\to\infty} \int_X f_n d\mu = \int_X f d\mu$.

## ãÂ. Product Measures and Fubini's Theorem

Let $(X, \mathcal{A}, \mu)$ and $(Y, \mathcal{B}, \nu)$ be measure spaces. The product measure $\mu \times \nu$ is defined on $\mathcal{A} \otimes \mathcal{B}$.

**Tonelli's Theorem:** For $f \ge 0$, the iterated integrals equal the product integral.
**Fubini's Theorem:** If $\int |f| d(\mu \times \nu) < \infty$, the iterated integrals are equal and equal the product integral.

## ãÈ. $L^p$ Spaces

The space $L^p(X, \mu)$ for $1 \le p < \infty$ consists of measurable functions such that:
$$\|f\|_{L^p} = \left( \int_X |f|^p d\mu \right)^{1/p} < \infty$$

For $p=\infty$, $\|f\|_{L^\infty} = \text{ess } \sup_{x \in X} |f(x)|$. These are Banach spaces (complete normed vector spaces).

### 4.1 Duality

The Riesz Representation Theorem states that for $1 \le p < \infty$, the dual space $(L^p)^*$ is isometric to $L^q$, where $1/p + 1/q = 1$. Every continuous linear functional $L$ on $L^p$ can be represented as $L(f) = \int fg d\mu$ for some unique $g \in L^q$.

## ãÊ. Specialized Measures

### 5.1 Radon Measures

On a locally compact Hausdorff space, a Radon measure is a Borel measure that is outer regular for all Borel sets, inner regular for open sets, and finite on compact sets.

### 5.2 Hausdorff Measure

The Hausdorff measure $\mathcal{H}^s$ generalizes length and volume to arbitrary dimensions $s \ge 0$, allowing for the measurement of fractal sets and irregular boundaries. It is defined using coverings by sets of small diameter:
$$\mathcal{H}^s(E) = \lim_{\delta \to 0} \inf \left\{ \sum_{i=1}^\infty \text{diam}(U_i)^s : E \subseteq \bigcup U_i, \text{diam}(U_i) < \delta \right\}$$
