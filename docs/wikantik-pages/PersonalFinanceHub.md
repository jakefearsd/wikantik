# PersonalFinance Hub

The retirement-planning and index-fund-investing clusters cover what to do with money you have set aside for the long term. This cluster covers everything that comes before that — earning, saving, protecting, and managing money in the day-to-day, year-to-year sense. It is the foundation on which longer-horizon planning sits.

If you are looking for the entry point, start with [PersonalFinanceGuide](PersonalFinanceGuide). It frames the order in which the rest of the cluster is best read.
